# T1.2_statistical_yearbook
OpenMusE Statistical Yearbook. The OpenMusE Statistical Yearbook offers an assessment of the scope of the music sector as it is visible in official statistics by compiling available data for European Union countries for the years 2022, 2023, and 2024.
Directory:
Main-report branch contains the static text report submitted to the European Commission.
Annexes branch contains country-level tables.
Code branch contains R code needed to replicate the data management and analysis.
